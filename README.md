## Project Overview

A civic tech platform designed to improve communication between citizens and government officials. Using NLP classifiers, the system sorts citizen requests and generates summarized reports for public review.

## Core Features

- Citizen feedback collection forms categorizing municipal requests.
- NLP classifier sorting feedback by department (e.g., roads, sanitation).
- Dashboard showing request tracking, resolution times, and community hotspots.
- Summary generator condensing duplicate issues for review.
- Docker setup packaging app and analytics pipelines.

## Technical Flow & Execution

Citizens submit municipal feedback. The NLP classifier categorizes the request, updates the dashboard, and notifies the relevant department representative.
